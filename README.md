#Space Launches | React App

A simple React App that pulls data from an API and displays it. For each item displayed, more data is shown when clicked on.  

**Technologies used:** 

[x] HTML 
[x] CSS
[x] React.js (React Router) 
[x] styled-components
[x] APIs.
